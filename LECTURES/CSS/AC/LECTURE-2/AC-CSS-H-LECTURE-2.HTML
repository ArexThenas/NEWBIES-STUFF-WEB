TEXT PROPERTIES

  • TEXT-ALIGN PROPERTY
    • The text align property is used to align text with respect to the webpage...
        Example : 
                  selector{
                                 text-align : center;
                  }

Note : center is one of the values for the text-align property,
       some of the values are as follows :
                                               1. Left
                                               2. Right
                                               3. Center
                                               4. Start
                                               5. End
  • TEXT-DECORATION PROPERTY
    • The text decoration property is used to add or remove decorative lines to the text on a webpage...
        Example : 
                  selector{
                                text-decoration : underline;
                  }

Note : some of the values are as follows :
                                               1. Underline
                                               2. Overline
                                               3. Line-through
                                               4. None

Extra : You can add extra values to a property like color and stuff...

            Example :
                     
                     selector{
                                text-decoration : line-through red;
                     }
  • FONT-WEIGHT PROPERTY
    • The font-weight property is used to control the boldness of text on a webpage...
        Example : 
                      selector{
                                  font-weight : 900;
                      }

Note : The value of this property can either be numerical (between 100 and 900) and 
       some of the values are as follows :
                                               1. Lighter
                                               2. Normal
                                               3. Bold
                                               4. Bolder
  • FONT-FAMILY PROPERTY
    • The font-family property is used to control the font of the text on a webpage...
            Example : 
                      selector{
                                  font-family : arial, roboto;
                      }

Note : You can either use a generic font family or a specific one...
        Figure out what these families are all about using the internet...

        Some of the values are as follows :
                                               1. Arial 
                                               2. Roboto 
                                               3. Serif 

Extra : I listed two values in the example as a fallback mechanism,
        what is a fallback mechanism you may ask? well search it yourself!
  • FONT-SIZE PROPERTY
    • The font-size property is used to set the font size of the text on a webpage...
            Example : 
                      selector{
                                  font-size : medium;
                      }

Note : Some of the values are as follows :
                                               1. Small
                                               2. Smaller
                                               3. Medium
                                               4. Large
                                               5. Larger
  • LINE-HEIGHT PROPERTY
    • The line-height property is used to set how much space or height the text on a webpage takes...
            Example : 
                      selector{
                                  line-height : normal ;
                      }

Note : You can use units or the text based value...
        Some of the values are as follows :
                                               1  Unset
                                               2. Inherit
                                               3. Normal
  • TEXT-TRANSFORM PROPERTY
    • The text-transform property is used to set the case of the text on a webpage...
            Example : 
                      selector{
                                  text-transform : uppercase ;
                      }

Note : Some of the values are as follows :
                                               1  Uppercase
                                               2. Lowercase
                                               3. Capitalize
                                               4. None

    UNITS IN CSS

    In CSS there are two types of units, absolute and relative...
These units are used for measuring and setting the size in a property...
    • ABSOLUTE UNITS
      • Absolute units are fixed units that are used for size related properties...
            Example : 
                      selector{
                                  text-size : 2px ;
                      }

Note : Some of the values are as follows :
                                               1  Px
                                               2. Cm

Extra : 96p = 1 inch
