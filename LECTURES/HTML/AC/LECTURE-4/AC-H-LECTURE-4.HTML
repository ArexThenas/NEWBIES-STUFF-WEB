LISTS IN HTML

In the lecture they two types of lists were mentioned:
        1. Ordered List: 
            - Used for items that have a specific order or sequence.
            - Created using the <ol> tag.
            - Each item is defined with the <li> tag.

            Example:
            
                <Ol>
                    <li>Apple</li>
                    <li>Mango</li>
                    <li>Orange</li>
                </ol>
            
        2. Unordered List:
            - Used for items that do not have a specific order.
            - Created using the <ul> tag.
            - Each item is defined with the <li> tag.

            Example:
            
                <Ul>
                    <li>Apple</li>
                    <li>Mango</li>
                    <li>Orange</li>
                </Ul>
            

Note: In unordered lists bullet points are used, whereas in ordered lists numbers or letters are used to indicate the order of items.

TABLES IN HTML

Tables are used to display data in a structured format with rows and columns.
    
        Example of a simple table:
    
        <table>
            <tr> 
                <th>Header 1</th> 
                <th>Header 2</th>
            </tr>
            <tr>
                <td>Data 1</td> 
                <td>Data 2</td>
            </tr>
             <tr>
                <td>Row 2, Data 1</td> 
                <td>Row 2, Data 2</td>
            </tr>
        </table>
    

Here, tr represents table rows,
      th represents table header,  
      td represents table data,
      table is used to contain previously mentioned elements.

CAPTION IN TABLES

The <caption> tag is used to provide a title or description for the table.
It should be placed immediately after the <table> tag.
        
        Example:
        
            <table>
                <caption>My Table Caption</caption>
                <tr>
                    <th>Header 1</th>
                    <th>Header 2</th>
                </tr>
                <tr>
                    <td>Data 1</td>
                    <td>Data 2</td>
                </tr>
            </table>

THEAD AND TBODY

Thead is used to group the header content in a table, while tbody is used to group the body content.
Tbody is useful for styling and scripting purposes, especially when dealing with large tables.
        
        Example:
        
            <table>
                <thead>
                    <tr>
                        <th>Header 1</th>
                        <th>Header 2</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>Data 1</td>
                        <td>Data 2</td>
                    </tr>
                </tbody>
            </table>

FORMS IN HTML

Forms are used to collect data from the users.

        Example of a simple form:
        
            <form>
                form elements go here
            </form>

ACTION ATTRIBUTE

Action attribute in for is used to define where the form data should be sent when the form is submitted.
it specifies the URL of the server-side script that will process the form data.

        Example:
        
            <form action="submit_form.php">
                Form elements go here
            </form>
        
You can also use a relative URL, like action="submit.php", which will submit the form to the same directory as the current page.

FORM INPUT ELEMENT

This element is used to enable the users to inpput data into the form.

        Example:
        
            <form action="submit_form.php">
                <input type="text" name="username" placeholder="Enter your username">
                <input type="submit" value="Submit">
            </form>

Here, the type attribute specifies the type of input field (text, password, checkbox, etc.),
the name attribute is used to identify the input field when the form is submitted,
and the placeholder attribute provides a hint to the user about what to enter in the field.

LABEL

The label element is used to associate a text description with an input element, provides a larger clickable area...

        Example:
        
            <form action="submit_form.php">
                <label for="username">Username:</label>
                <input type="text" id="username" name="username" placeholder="Enter your username">
                <input type="submit" value="Submit">
            </form>
        

Here, if the for attribute is used then one must use an id in the input elements to associate them...

CLASS AND ID

ID      :    An id is a unique identifier that should ideally be used uniquely for each html element...
CLASS   :    An attribute that can be used to group different id's or elements... 
    
            Example :
                    
                        <p id = "id1" class = "group1">...</p>
                        <p id = "id2" class = "group2">...</p>
                    
These can be preety useful when using CSS...

CHECKBOX

Checkbox is used to provide multiple option to select from... 

        Example :
                
                    <label for = "id1"> Label
                            <input type = "checkbox" id = "id1">
                    </label>
                

                or

                 
                    <label for = "id1"> Label </label>
                            <input type = "checkbox"  id = "id1">
                
I tried using both methods and appearantly they both worked... 

    1.Using the former is less flexible since both the label and input need to be together,
    here using the "for" attributre is optional since the label tag is wrapping the input element...

    2.Using the latter is more flexible since label and input contain the same id therefore they can be multiple lines apart and still work...

SELECT AND OPTION TAGS

The select and option tags are used simultaneously, the select tag is used to wrap multiple options 
    and to provide the user with a dropdown menu...

            Example : 
                     
                        <label for="cars">Choose a car:</label>
                            <select id="cars" name="car">
                                <option value="volvo">Volvo</option>
                                <option value="bmw">BMW</option>
                                <option value="audi">Audi</option>
                            </select>

IFRAME TAG

Well this is an impressive tag for sure! 
Why? You may ask, well this tag lets you display other websites on your webpage!

            Example : 
                     
                        <iframe src="link"></iframe>
                     

Here, if you place some kind of text between the opening and closing tags
      it will be treated as the fallback only when the users browser does not support iframe tag 
      or the iframe tag is disabled/blocked in the browser...

VIDEO TAG

The video tag is used to add videos to a webpage either through an absolute link or a relative one...

            Example : 
                     
                        <video src="link">Text goes here </video>
                     

You can enter some text between the opening tag and the closing tag as a fallback,
just like you did in the iframe tag...

Note : Both the iframe tag and video tag may not work properly and display the fallback 
       when a site does not allow for itself to be displayed using the iframe tag,
       or when your network is down...

       You can always try experimenting with the tags to learn more also use ai for more clarity...
