HTML LECTURE-1

HTML

Html stands for "HYPER TEXT MARKUP LANGUAGE", it is widely used for structuring a webpage and its contents.
the components used to make the structure of a website are called html tags.

    Extension for an  html file is ".html".

By default "index.html" is treated as the homepage, unless you manually specify which file to use as homepage "index.html" will be used by the server.

HTML TAGS

An html tag is a container for some content or other html tags. 

    For example:  <p>This is a paragraph.</p>

Here, the opening tag is - <p> ,
      the closing tag is - </p> ,
      the content is     -"This is a paragraph." ,
      the element is     -  <p>This is a paragraph.</p>

BASIC HTML PAGE

<!DOCTYPE html>         
    <html>                         
        <head>                     
            <title>                
        </head>
        <body>                     
            <p></p>          
        </body>
    </html>

QUICK POINTS

         a. Html tag is the parent tag of head and body.
         b. Most of the elements have an opening and closing tag with content in between.
         c. Some tags have no content in between, or no closing tag.
         d. You can use inspect code in browser to edit html code though it won't be reflected in the code editor.
         e. Html is case insensetive.

COMMENTS IN HTML

A comment is that part of the code which should not be parsed on the browser. 
Comments are helpful when you are collaborating with someone or writing a lot of lines of code.
Use + in windows to comment out a line of code, in an empty line a blank comment will form.

    An example of what a comment looks  like - <!-- This is a comment. -->

BASIC HTML TAGS

In this lecture basic html tags and attributes have been covered,

HTML ATTRIBUTES

Html attributes can be used to do some of the work done by css inside the tag itself,
though it would be a better idea to use a css file for styling the page.
    
           For example : <html lang="en">

Here, the attribute is "lang" and its value is "en", which tells the browser that the language of the page is English.

HEADING TAGS

Heading tags as the name suggests are used to add headings on a webpage,
there are six types of heading tags from h1 to h6, these tags get smller as the number increases.
    
The six heading tags are written as follows -
             <h1>Heading 1</h1> 
             <h2>Heading 2</h2> 
             <h3>Heading 3</h3> 
             <h4>Heading 4</h4> 
             <h5>Heading 5</h5> 
             <h6>Heading 6</h6> 
    
It is to be noted that h1 should only be use once per page as it is the main heading of the page,
and the rest of the headings can be used as per requirement.

PARAGRAPH TAG

The paragraph tag is used to add a paragraph to the page.
               Example : <p>This is a paragraph. </p> 
    
paragraph tag automatically sorts the text by removing extra spaces and line breaks.

ANCHOR TAG

The anchor tag is used to add relative and absolute links to the page using href attribute.
you can also add text to the link which will be displayed on the page.
    
            Example : 
                        <a href="link"> site name </a>                 (for absolute links.)
                        <a href="folder/file.html"> file name </a>     (for relative links.)

Here, the href attribute is used to specify the link and the text "site name" will be displayed on the page.

IMAGE TAG

The image tag is used to display images on a webpage by eityer using a relative or absolute link.
    
            Example : 
                        <img src="link" alt="image from web";                 (for absolute links.)
                        <img src="folder/file.html" alt="local image">        (for relative links.)
    
Here, the src attribute is used to specify the link of the image,
the alt attribute specifies the text to be displayed if the image is not loaded.

BR TAG

The br tag is used to add a line break in the text.
    
            Example : <br>

This tag is commonly used to add lines in a paragraph or to add a line break in the text.

BOLD, ITALIC AND UNDERLINE TAGS

 The bold, italic and underline tags are used to style the text on a webpage.
    
            Example : 
                        <b>Bold</b>
                        <i>Italic;</i>
                        <u>Underline</u>

These tags are used to make the text bold, italic and underlined respectively.

BIG AND SMALL TAGS

        The big and small tags are used to change the size of the text on a webpage.
    
            Example : 
                        <big>Big</big>
                        <big>Big</big>

It can be understood by self that the big tag makes the text bigger and the small tag makes the text smaller.

HR TAG

The hr tag is quite useful to add a horizontal line in the text,
it is commonly used to separate sections of a webpage.

            Example : <hr>

As you can see I personally have been using this tag to make sections on the html lecture pages.

SUBSCRIPT AND SUPERSCRIPT TAG

The subscript and superscript tags are used to add subscript and superscript text on a webpage.

        Example : 
                        <sub>Subscript</sub>
                        <sup>Superscript</sup>

You can use these tags to make text appear above or below on the line of text.
These tags can be used for various purposes like writing chemical formulas or mathematical equations.

PRE TAG

The pre tag is used to display text as it is i.e. without ignoring spaces or line breaks
            Example : <pre> This is preformatted text. </pre>

TYPES OF TAGS

There are two types of tags in context of seo.
            
            1. semantic tags
                Definition:
                    Tags that describe the meaning and purpose of the content inside them.

                Purpose:
                    Provide context to browsers, search engines, and developers about the role of the content.

                Examples:
                    <header>, <footer>, <article>, <section>, <nav>, <main>, <aside>, <figure>

                Benefit:

                    Improves SEO (search engines understand page structure).

                    Enhances accessibility for screen readers.

                    Makes the HTML more readable and maintainable


            2. non-semantic tags

                Definition:
                    Tags that do not convey any meaning about the content they contain; they are only for structure and styling.

                Purpose:
                    Used as generic containers or wrappers for grouping elements when no specific meaning applies.

                Examples:
                    <div> (block-level container), <span> (inline container).

                Drawback:

                    No built-in meaning, so search engines and assistive technologies get less context.

                    Code is harder to understand without extra classes or IDs.

LAYOUT

        Here for layout I am talking about the structure of a web page.
        In the lecture they talked about the following layout tags:

        1. <header>: Represents the header section of a page, typically containing the site title, logo, and navigation links.

        2. <main>: Represents the main content area of a page, where the primary content is displayed.

                a. <section>: Represents a thematic grouping of content, typically with a heading.

                b. <article>: Represents a self-contained piece of content, such as a blog post or news article.

                c. <aside>: Represents content that is tangentially related to the main content, such as sidebars or related links.

        3. <footer>: Represents the footer section of a page, usually containing copyright information, links, and other metadata.

REVISITING ANCHOR TAG

        If you use attribute target="_blank" in an anchor tag, it opens the link in a new tab or window.
        This is useful for keeping users on your site while allowing them to view external content.

        Example:
                <a href="link" target="_main"> Text </a>
                
                Note : If you replace "text" with an image, it will still work.

REVISITING IMAGE TAG

        You can use height and width attributes in the image tag to control the size of the image.
        
        Example:    
                <img src="image.jpg" alt="description" height="200" width="300">

DIV TAG

        The <div> tag is a block-level element used to group content together.
        It is often used for layout purposes and can be styled with CSS.

        Example:
                <div class="container"> Content here </div>

        Note: The <div> tag does not have any semantic meaning, so it should be used when no other semantic tag fits the purpose.

SPAN TAG

        span tag is also a sort of div tag but it is an inline element.
        It is used to apply styles or scripts to a specific part of text without breaking the flow of the content.

LISTS IN HTML

In the lecture they two types of lists were mentioned:
        1. Ordered List: 
            - Used for items that have a specific order or sequence.
            - Created using the <ol> tag.
            - Each item is defined with the <li> tag.

            Example:
            
                <Ol>
                    <li>Apple</li>
                    <li>Mango</li>
                    <li>Orange</li>
                </ol>
            
        2. Unordered List:
            - Used for items that do not have a specific order.
            - Created using the <ul> tag.
            - Each item is defined with the <li> tag.

            Example:
            
                <Ul>
                    <li>Apple</li>
                    <li>Mango</li>
                    <li>Orange</li>
                </Ul>
            

Note: In unordered lists bullet points are used, whereas in ordered lists numbers or letters are used to indicate the order of items.

TABLES IN HTML

Tables are used to display data in a structured format with rows and columns.
    
        Example of a simple table:
    
        <table>
            <tr> 
                <th>Header 1</th> 
                <th>Header 2</th>
            </tr>
            <tr>
                <td>Data 1</td> 
                <td>Data 2</td>
            </tr>
             <tr>
                <td>Row 2, Data 1</td> 
                <td>Row 2, Data 2</td>
            </tr>
        </table>
    

Here, tr represents table rows,
      th represents table header,  
      td represents table data,
      table is used to contain previously mentioned elements.

CAPTION IN TABLES

The <caption> tag is used to provide a title or description for the table.
It should be placed immediately after the <table> tag.
        
        Example:
        
            <table>
                <caption>My Table Caption</caption>
                <tr>
                    <th>Header 1</th>
                    <th>Header 2</th>
                </tr>
                <tr>
                    <td>Data 1</td>
                    <td>Data 2</td>
                </tr>
            </table>

THEAD AND TBODY

Thead is used to group the header content in a table, while tbody is used to group the body content.
Tbody is useful for styling and scripting purposes, especially when dealing with large tables.
        
        Example:
        
            <table>
                <thead>
                    <tr>
                        <th>Header 1</th>
                        <th>Header 2</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>Data 1</td>
                        <td>Data 2</td>
                    </tr>
                </tbody>
            </table>

FORMS IN HTML

Forms are used to collect data from the users.

        Example of a simple form:
        
            <form>
                form elements go here
            </form>

ACTION ATTRIBUTE

Action attribute in for is used to define where the form data should be sent when the form is submitted.
it specifies the URL of the server-side script that will process the form data.

        Example:
        
            <form action="submit_form.php">
                Form elements go here
            </form>
        
You can also use a relative URL, like action="submit.php", which will submit the form to the same directory as the current page.

FORM INPUT ELEMENT

This element is used to enable the users to inpput data into the form.

        Example:
        
            <form action="submit_form.php">
                <input type="text" name="username" placeholder="Enter your username">
                <input type="submit" value="Submit">
            </form>
            
Here, the type attribute specifies the type of input field (text, password, checkbox, etc.),
the name attribute is used to identify the input field when the form is submitted,
and the placeholder attribute provides a hint to the user about what to enter in the field.

LABEL

The label element is used to associate a text description with an input element, provides a larger clickable area...

        Example:
        
            <form action="submit_form.php">
                <label for="username">Username:</label>
                <input type="text" id="username" name="username" placeholder="Enter your username">
                <input type="submit" value="Submit">
            </form>
        

Here, if the for attribute is used then one must use an id in the input elements to associate them...

CLASS AND ID

ID      :    An id is a unique identifier that should ideally be used uniquely for each html element...
CLASS   :    An attribute that can be used to group different id's or elements... 
    
            Example :
                    
                        <p id = "id1" class = "group1">...</p>
                        <p id = "id2" class = "group2">...</p>
                    
These can be preety useful when using CSS...

VIDEO TAG

just like you did in the iframe tag...

Note : Both the iframe tag and video tag may not work properly and display the fallback
